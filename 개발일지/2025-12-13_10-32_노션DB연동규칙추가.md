## ✅ 해결된 것
### 문제 1: 노션 DB 연동 규칙 추가
- **해결 방법**: `.cursor/rules/development-journal.mdc`에 노션 DB URL, Data Source ID, 타이틀 속성명(`이름`) 및 MCP 실행 절차를 명시하여 “로컬 md + 노션 페이지” 동시 기록을 강제
- **관련 파일**: `.cursor/rules/development-journal.mdc`

### 문제 2: 노션 DB 스키마 확인
- **해결 방법**: 노션 MCP `fetch`로 DB의 Data Source ID와 스키마(타이틀 속성명)를 확인하여 규칙에 반영
- **관련 DB**: `https://www.notion.so/2c8e3e75d754805a8d24ec86eebd0a37?v=2c8e3e75d7548025a25b000c878194be&source=copy_link`

## ❌ 해결되지 않은 것
### 노션 DB 속성 확장(상태/태그/날짜 등)
- **이유**: 현재 DB는 타이틀(`이름`)만 존재하여, 향후 속성 확장 시 규칙 업데이트가 필요
