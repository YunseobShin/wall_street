# 당신이 잠든 사이 (While You Were Sleeping)
## 미국주식 데일리 브리핑 서비스 기획서

---

## 1. 서비스 개요

**당신이 잠든 사이**는 한국인 투자자를 위한 미국주식 자동 데일리 브리핑 서비스입니다. 한국 시간으로 밤~새벽에 진행되는 미국 증시의 핵심 움직임을 AI가 자동으로 분석하여, 아침 출근길에 한눈에 파악할 수 있는 이미지 브리핑으로 제공합니다. Yahoo Finance의 실시간 거래량과 상승률 데이터를 기반으로 가장 화제가 된 종목을 자동 선정하며, 별도의 사용자 입력 없이 매일 아침 이메일 또는 슬랙으로 브리핑이 자동 발송됩니다. **"복잡한 분석은 AI에게, 투자 결정은 당신에게"**라는 철학 아래, 바쁜 직장인 투자자의 정보 격차를 해소합니다.

## 2. 핵심 가치 제안 (Value Proposition)

### 🎯 핵심 가치

| 구분 | 기존 방식 | 당신이 잠든 사이 |
|------|----------|------------------|
| **정보 수집** | 여러 앱/사이트 직접 확인 | 자동 수집 & 큐레이션 |
| **종목 선정** | 직접 관심 종목 설정 필요 | 화제 종목 자동 선정 |
| **분석 시간** | 30분~1시간 소요 | 3분 이미지 브리핑 |
| **정보 접근** | 능동적 검색 필요 | 수동적 수신 (Chatless) |
| **언어** | 영문 뉴스 해석 필요 | 한글 요약 제공 |

### 💡 핵심 차별점

1. **Chatless Experience**: 사용자 입력 없이 100% 자동화된 브리핑
2. **데이터 기반 종목 선정**: 감이 아닌 거래량/상승률 기반 객관적 선정
3. **시각적 브리핑**: 텍스트가 아닌 한눈에 파악 가능한 이미지 형태
4. **시간대 최적화**: 한국 아침 시간에 맞춘 발송으로 출근길 소비 가능

## 3. 주요 기능 목록

### 🔧 Core Features

| # | 기능명 | 설명 | 우선순위 |
|---|--------|------|----------|
| 1 | **화제 종목 자동 선정** | Yahoo Finance API 기반 거래량 상위 + 상승률 상위 종목 자동 추출 | P0 |
| 2 | **종목 정보 수집** | 선정된 종목의 기본 정보(시가총액, PER, 섹터 등) 자동 수집 | P0 |
| 3 | **관련 뉴스 큐레이션** | 각 종목별 최신 뉴스 2-3개 자동 수집 및 한글 요약 | P0 |
| 4 | **AI 이미지 브리핑 생성** | 수집된 정보를 카드 뉴스 형태의 시각적 브리핑으로 자동 생성 | P0 |
| 5 | **멀티채널 자동 발송** | 이메일 / 슬랙 / 카카오톡 등 선호 채널로 자동 발송 | P0 |
| 6 | **시장 전체 요약** | 주요 지수(S&P500, 나스닥, 다우) 일일 변동 요약 | P1 |
| 7 | **주간 리포트** | 금요일 발송되는 한 주 종합 브리핑 | P2 |

### 📊 기능 상세

#### 3-1. 화제 종목 자동 선정
```
선정 기준:
├── 거래량 상위 5종목 (Volume Leaders)
├── 상승률 상위 3종목 (Top Gainers)  
├── 하락률 상위 2종목 (Top Losers)
└── 총 10개 종목 / 일
```

#### 3-2. AI 이미지 브리핑 구성
```
브리핑 이미지 구성:
├── 커버: 오늘의 시장 한줄 요약
├── 종목 카드 (종목당 1장)
│   ├── 종목명 / 티커 / 로고
│   ├── 주가 변동 (%, $)
│   ├── 거래량 변화
│   ├── 핵심 뉴스 1줄 요약
│   └── AI 인사이트 (왜 움직였나)
└── 마무리: 내일 주목할 이벤트
```

## 5. 성공 지표 (KPI)

### 📈 핵심 KPI 대시보드

| # | 지표명 | 정의 | 목표 (3개월) | 측정 방법 |
|---|--------|------|-------------|-----------|
| 1 | **DAU (일간 활성 사용자)** | 브리핑을 열어본 고유 사용자 수 | 1,000명 | 이메일 오픈율 + 슬랙 조회 |
| 2 | **이메일 오픈율** | 발송 대비 오픈 비율 | 45% 이상 | 이메일 트래킹 |
| 3 | **구독 유지율 (Retention)** | 4주 후에도 구독 유지 비율 | 70% 이상 | 구독 취소율 역산 |
| 4 | **NPS (순추천지수)** | "이 서비스를 추천하시겠습니까?" | 50점 이상 | 월 1회 서베이 |
| 5 | **공유율** | 브리핑을 SNS/지인에게 공유한 비율 | 10% 이상 | 공유 버튼 클릭 추적 |

### 🎯 단계별 목표

```
[Phase 1: MVP] 1-2개월
├── 목표: 서비스 검증
├── 구독자 500명 확보
├── 이메일 오픈율 40% 달성
└── 핵심 기능 안정화

[Phase 2: Growth] 3-4개월  
├── 목표: 성장 가속
├── 구독자 3,000명 확보
├── 슬랙/카카오톡 채널 추가
└── 주간 리포트 기능 출시

[Phase 3: Monetization] 5-6개월
├── 목표: 수익화
├── 프리미엄 플랜 출시 (섹터별 브리핑)
├── 유료 전환율 5% 달성
└── 월 구독 모델 안정화
```
